# Fase 1

Fase 1 documenta `edugo-shared` como un sistema independiente. La regla de trabajo en esta etapa es simple: solo se describe lo que se puede observar dentro de esta carpeta.

## Incluye

- Estructura modular real (`go.mod` por modulo).
- Procesos observables en codigo, tests, `Makefile` y workflows.
- Arquitectura local por modulo.
- Estado operativo actual del repositorio.

## Excluye

- Integraciones con el ecosistema externo.
- Contratos de uso vistos desde otros repositorios.
- Estandares futuros de release por modulo aun no implementados.

## Orden de lectura sugerido

1. [Arquitectura del repositorio](repository-architecture.md)
2. [Mapa de procesos](process-landscape.md)
3. [Catalogo de modulos](module-catalog.md)
4. Documentacion propia de cada modulo desde su carpeta
